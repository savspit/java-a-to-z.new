Пакет:
 ru.shestakov.start
Классы:
    Tracker - основной класс, содержащий public static void main(String[] args)
	
Пакет:
 ru.shestakov.models
Классы:
    User - получает и устанавливает имя автора
	UserStore - добавляет пользователя в список пользователей
	Item - основной класс для заявок, изменение заявки
	Task - класс, наследованный от Item, работа с заявками будет осуществляться через него
	TaskStore - добавляет заявку в список заявок, показывает весь список заявок, показывает список заявок по отбору, находит / удаляет заявку

	
Приложение позволяет добавлять/изменять/удалять заявки, 
показывать список заявок, либо список заявок по отбору.
